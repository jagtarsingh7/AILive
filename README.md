# canvass-api-model-store
FastAPI backend for Canvass AI Live

## Instructions
### Run the project
```bash
cd canvass-api-model-store
make run
```
### Run tests
```bash
make test
```
### Run tests with coverage
```bash
make test-cov
```
### Run tests with coverage and HTML report
```bash
make test-cov-html
```

### Add a new dependency
```bash
make add
```

### Add a new dev dependency
```bash
make add-dev
```

### Remove a dependency
```bash
make remove
```
